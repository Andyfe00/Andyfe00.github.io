# Andyfe00.github.io

This portfolio was built using the following tools:

- **Bootstrap 5.3.3** — Responsive grid system, layout and components.
- **Swiper.js 11** — Interactive carousel for the portfolio section.
- **Typed.js 2.0.12** — Animated typing effect on the home section.
- **TagCanvas.js** — 3D rotating skills cloud.
- **Font Awesome 6.5.0** — Icons for social media and UI.
- **Google Fonts** — For custom typography.
